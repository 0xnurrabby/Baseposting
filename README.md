## `{ quick_start }`

```bash
git clone https://github.com/0xnurrabby/Baseposting
cd Baseposting
npm install
npm run dev
```

> **Required env vars** Гўв‚¬вЂќ create a `.env` file:

```env
VITE_TIP_RECIPIENT=0xYourWalletAddress
UPSTASH_REDIS_REST_URL=your_upstash_url
UPSTASH_REDIS_REST_TOKEN=your_upstash_token
```

> **Deploy to Vercel:**

```bash
npm run build
# push to GitHub and import in vercel.com
# Vercel auto-detects vite + runs "npm run build" ГўвЂ вЂ™ outputs /dist
```
